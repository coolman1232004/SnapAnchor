# SnapAnchor 2.1.0

SnapAnchor 2.1.0 improves long scrolling capture and refreshes the application with a calm warm-neutral interface.

## Highlights

- Long capture now directs scrolling to the window underneath the selected region and falls back to system wheel input when needed.
- A clear message is shown when the selected content cannot scroll; a normal viewport is no longer returned as a successful long capture.
- The dashboard, Capture History, Preferences, capture controls, and pin surfaces now share a warm off-white, white, charcoal, and warm-grey design.
- The app, taskbar, and tray icon now use a flat charcoal capture-frame and anchor symbol.
- Capture History includes a **Back to SnapAnchor** button for returning to the main dashboard.

## Downloads

- **Installer:** `SnapAnchor-Setup-win-x64.exe`
- **Portable:** `SnapAnchor-Portable-win-x64.zip`

The installer and portable build are currently unsigned, so Windows SmartScreen may display a warning. SnapAnchor requires 64-bit Windows 10 version 1809 or later, or Windows 11.
